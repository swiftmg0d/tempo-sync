# @tempo-sync/shared

Shared middleware, error types, utilities, and API helpers used across `@tempo-sync/api` and `@tempo-sync/integrations`.

## Exports

- **App factory** — `createApp()` with CORS, logger, and health endpoint
- **Middleware** — CORS, error handler, rate limiter, validation
- **Errors** — Custom error types
- **Utils** — Crypto and date helpers
