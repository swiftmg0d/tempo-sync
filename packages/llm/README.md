# @tempo-sync/llm

LLM integration layer for Tempo-Sync. Supports multiple providers for generating activity insights, descriptions, and track leaderboard analysis.

## Providers

OpenAI · Groq · OpenRouter · SambaNova · Cerebras

## Exports

- **Chat** — Unified chat completion interface across providers
- **Prompts** — Activity description, activity insight, and track leaderboard prompts
- **Config** — Provider configuration and model selection
